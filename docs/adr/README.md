# Architecture Decision Records

Store ADRs in this directory using sequential numbering:

- `0001-short-title.md`
- `0002-short-title.md`

Keep ADRs short. A single paragraph is often enough.

Recommended shape:

```md
# Decision Title

1-3 sentences describing the context, the decision, and why it was chosen.
```

Optional sections are allowed only when they add clear value:

- considered options
- consequences
- status
